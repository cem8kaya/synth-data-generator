About Generic Synthetic Data Generator

Multi-Domain Time-Series Data Generation System

Overview

The Generic Synthetic Data Generator is a domain-agnostic system for generating realistic synthetic time-series data with configurable patterns, correlations, anomalies, and temporal dynamics.

Version 2.0 has been completely refactored from a telecom-specific tool into a multi-domain platform that supports:

  • Telecommunications
  • Financial Services
  • Healthcare
  • Manufacturing
  • E-commerce
  • IoT/Sensors
  • Custom Domains
Key Features:
  • 6+ pre-built domain templates
  • 7 probability distributions
  • Correlation modeling
  • Temporal patterns
  • Anomaly injection

Key Features
Statistical Foundation
  • Normal Distribution: General-purpose metrics
  • Gamma Distribution: Latency, delays, processing times
  • Lognormal Distribution: Throughput, revenue, data volume
  • Beta Distribution: Rates, percentages, success rates
  • Poisson Distribution: Event counts, arrivals
  • Exponential Distribution: Wait times, inter-arrival times
  • Uniform Distribution: Random values in range
Advanced Capabilities
  • Correlation Modeling: Gaussian copula-based correlation preservation
  • Seasonality: Daily, weekly, and custom periodic patterns
  • ARIMA Smoothing: Temporal coherence and autocorrelation
  • Change Points: System upgrades, configuration changes
  • Anomaly Injection: Spikes, drops, outages, degradation
  • Multi-Entity Systems: Complex dependencies between components
System Architecture 
┌─────────────────────────────────────┐
│   Domain Templates Layer            │
│   (Telecom|Finance|Healthcare|...)  │
├─────────────────────────────────────┤
│   Generic Configuration Layer       │
│   (Entities, Metrics, Schemas)      │
├─────────────────────────────────────┤
│   Core Generation Engine            │
│   (Distributions, Correlations)     │
├─────────────────────────────────────┤
│   Infrastructure Layer              │
│   (API, Storage, Validation)        │
└─────────────────────────────────────┘
Layer Description:
  • Domain Templates: Pre-configured industry-specific setups
  • Configuration: Generic schemas for entities, metrics, patterns
  • Core Engine: Statistical generation, correlation, temporal processing
  • Infrastructure: REST API, file I/O, validation framework
Technical Specifications
Performance
Generation Speed ~1,000 windows/min
Memory Usage <2 GB
Startup Time <2 seconds
API Response <100ms (excluding generation)
Technology Stack
  • Backend: Python 3.8+, Flask
  • Data Processing: NumPy, Pandas, SciPy
  • Frontend: Bootstrap 5, Vanilla JS
  • Output Formats: CSV, Parquet, JSON
  • API: RESTful HTTP/JSON
API Endpoints
Method Endpoint Description
GET /api/templates List all available domain templates
GET /api/template/<domain> Get specific domain template
POST /api/generate Generate synthetic data
POST /api/validate Validate configuration
GET /api/health Health check
GET /api/info System information
GET /api/download/<filename> Download generated file
